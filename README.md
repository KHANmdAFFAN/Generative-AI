# Generative AI Project

This repository contains projects and experiments related to **Generative AI**.
The goal of this repository is to explore modern AI technologies such as **Large Language Models (LLMs)**, prompt engineering, and AI-powered applications.

## 🚀 Features

* Experiments with Generative AI models
* Example projects and code implementations
* Prompt engineering examples
* AI automation scripts

## 🛠 Technologies Used

* Python
* OpenAI API / LLMs
* Machine Learning libraries
* Git & GitHub

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/KHANmdAFFAN/Generative-AI.git
```

Go to the project folder:

```
cd Generative-AI
```

Install dependencies:

```
pip install -r requirements.txt
```

## ▶️ Usage

Run the project using:

```
python main.py
```
